# Travel & Tourism Website

A responsive frontend website for exploring and planning trips to various destinations.  
Built using **HTML, CSS, and JavaScript**, with an attractive design and user-friendly interface.

---

## **Features**
- Browse popular travel destinations
- View destination details (images, description)
- Responsive layout for desktop and mobile
- Interactive elements like navigation menus, buttons, and image galleries
- Contact form for inquiries (static frontend)
- Smooth scrolling and animations for better user experience

---

## **Technology Stack**
- **Frontend:** HTML5, CSS3, JavaScript
- **Styling:** CSS Flexbox/Grid, optionally Bootstrap
- **Responsive Design:** Works on desktops, tablets, and mobile devices

---

## **Installation & Usage**
1. Clone the repository:

```bash
git clone https://github.com/sheikahmadalisha/Travel-And-Tourism-Frontend.git
```
2. Navigate to the project folder:
```bash
cd Travel-And-Tourism-Frontend
```
3. Open index.html in your browser to view the website.
